# SentimentalAnalysis 🤖

## AI-Powered Sentiment Analysis System for SMEs

### Description
Machine learning system that analyzes customer sentiment 
from text feedback and images to help small businesses 
understand customer needs and improve their services.

### Features
- Text sentiment classification
- Customer feedback analysis
- CSV data processing
- PWA (Progressive Web App)

### Tech Stack
- Python
- Machine Learning / NLP
- Flask
